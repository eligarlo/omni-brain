# Omni Brain

## Backend Setup

1. Install dependencies:
   `cd api && pip install -r requirements.txt`
2. Update the `.env` file with your OpenAI and Langfuse credentials.
3. Start the server:
   `fastapi dev`

## Frontend Setup

1. Install dependencies:
   `cd ui && npm install`
2. Start the development server:
   `npm run dev`
